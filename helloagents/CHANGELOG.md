## [Unreleased]

### 修复
- 修复 `tmp_stage_then_move` 模式在启用远端防重时仍可能先上传到 `/tmp/115helper_stage/...` 的问题：现在会先按最终目标目录执行同名/SHA1 检查，命中后直接跳过，不再进入临时目录。
- 修复 `tmp_stage_then_move` 模式对远端目录存在性的误判：当 `get_dir_id_by_path()` 未命中时，新增基于父目录子项列表的二次确认，避免已存在目录被错误当作缺失目录后转入临时上传。

- 新增 `tmp_stage_then_move` 执行方式：当目标目录在 115 中不存在时，先上传到 `/tmp/115helper_stage/...` 再整目录移动到最终位置，用于规避最近操作路径识别异常。
- 调整插件对齐模式下的上传后确认时序：改为按 Plugin 风格使用 2s/4s/8s 递增等待，减少“新建目录后最近操作未及时可见”的问题。
- 新增 `upload_flow_mode` 执行方式开关，默认按插件方式逐文件准备目录、上传后轮询确认，并保留旧版批处理缓存兼容模式。
- 新增《115 上传流程对比：115helper vs MoviePilot-Plugins》文档，明确两者在目录创建顺序、目录文件同步时机与上传 API 链路上的一致点与差异。
- 新增前端移动端适配：支持手机端抽屉导航、卡片列表、响应式表单与日志视图。
- 对齐 plugin `_get_folder` 语义：远端目录准备统一走“获取目录，不存在则创建”的目录对象入口。
- 新增按 plugin 方式预创建远端叶子目录的能力：上传前先收集所有叶子目录并批量确保目录存在。
- 新增 plugin 风格的 115 Open 上传链路：支持 `open/upload/init`、二次校验、上传凭证获取、OSS 分片上传与合并。
- 新增 Open 上传日志回调与取消检查点，后台任务可在分片阶段输出实时日志并响应取消。
- 新增 `P115_OPEN_ACCESS_TOKEN`、`P115_OPEN_REFRESH_TOKEN` 配置项，用于启用 Open 上传。
- 初始化知识库与首个方案包，规划“本地目录同步到 115 网盘”的前后端项目。
- 初始化 FastAPI + Vue 3 前后端项目骨架。

- 新增本地目录扫描、后缀过滤、115 秒传/分片上传策略与 APScheduler 调度能力。
- 新增多任务管理、任务调度状态快照与任务级执行日志能力。
- 新增基于 SSE 的前端实时滚动日志能力。
